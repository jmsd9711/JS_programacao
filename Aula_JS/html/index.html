java script

java script e uma linguagem criada na decada de 1990 voltada para a parte de desenvolvimento web. É uma linguagem do lado do cliente(client-side) e possui tipagem dinamica, ou seja, o tipo de vairavel e atribuido automaticamente.

Exercícios

  1. Elabore um programa em que solicite ao usuário dois números n1 e n2. Realize a subtração dos dois valores.
  2. Faça um exercício que realize a multiplicação entre 3 números
  3. Realize o cubo de um número digitado pelo usuário
  4. Faça um código que exiba o antecessor e o sucessor de um número digitado pelo usuário
  5. Faça um programa que realize o cálculo da área do triângulo
  6. Faça um programa em que o usuário digite a quantidade de KM percorridos e a quantidade de Litros gastos e apresente a média de km por litro (km/l)
  7. Faça um programa em que o usuário digite o valor da compra e a quantidade de parcelas e exiba o valor de cada parcela
  8. Faça um programa em que o usuário digite as 4 notas de um aluno e apresente a média anual deste aluno
  9. Faça um programa que o usuário digite o valor de uma compra e apresente o valor com 15% de acréscimo e o valor com 15% de desconto
